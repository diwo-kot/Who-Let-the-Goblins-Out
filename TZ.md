# Техническое задание для 3D-билдера (Roblox Studio)

## Проект: "Who Let the Goblins Out"

**Задача:** Построить **2 карты** — Лобби и Офис (1 карта).  
**Стиль:** Полу-мультяшный (Stylized / Toon). Пример стиля приложен скриншотом.  
**Формат сдачи:** .rbxl файлы (или готовые модели в Studio).

---

## 1. Лобби (Hub)

**Концепция:** Парящий остров в магическом измерении. Вдохновение — **Burglin' Gnomes** (яркие цвета, сказочная атмосфера, летающие камни/острова).

### Что построить:

| Объект | Описание |
|--------|----------|
| **Остров** | Парящая скала/земля с травой. Края неровные, висят корни/сталактиты снизу. |
| **Табло с правилами** | Большая доска/камень с текстом (можно просто место под текст, скрипты потом добавят). |
| **Зона спавна игроков** | 10 точек появления (SpawnLocation) на острове. |
| **Маленькие островки** | 2-3 крошечных островка вокруг главного (по ним летают частицы/светлячки, чисто для красоты). |
| **Декорации** | Грибы большие, светящиеся растения, заборчик из веток, облака (полупрозрачные сферы). |
| **Небо/фон** | Сделать красивый фон (скайбокс) — звёздное небо с разноцветными туманностями. |

**Размер:** ~150x150 (не слишком большой, чтобы игроки не терялись).

---

## 2. Офис (игровая карта)

**Концепция:** Обычный офис в полу-мультяшном стиле. Всё как в жизни, но с преувеличенными пропорциями (огромные мониторы, высокие столы и т.п.).

### 2.1. Комнаты (зоны)

| Комната | Описание |
|---------|----------|
| **Комната охраны** | Стеклянная будка/комната. Внутри — кресло, стол с мониторами (выключенными). Окна выходят в коридор.|
| **Общий опенспейс** | Основное пространство с рядами столов, стульев, компьютеров. |
| **Кабинет начальника** | Отдельная комната со стеклянными стенами. Внутри — большой стол, сейф, ковёр, стеллаж с папками. |
| **Кухня/столовая** | Столик, микроволновка, кофемашина, кулер с водой, раковина. |
| **Коридор** | Соединяет комнаты. Можно с ковровой дорожкой. |
| **Кладовка** | Маленькая тесная комната с коробками, шваброй, ведром.|
| **Вентиляция** | Соединяет комнаты, такие короткие проходы|

### 2.2. Укрытия для гоблинов (ОБЯЗАТЕЛЬНО)

| Укрытие | Как выглядит |
|---------|--------------|
| **Под стол** | Офисный стол уголок с ящиками |
| **В ящик стола** | Сделать в некоторых столах приоткрытый ящик|
| **За монитором** | Большие широкие мониторы, которые закрывают гоблина целиком (гоблин маленький). |
| **Кулер с водой** | Прозрачная 20-литровая бутыль. |
| **Цветочный горшок** | Большой горшок с зелёным кустом/цветком|


### 2.3. Дополнительные объекты (декор)

| Объект | Примечание |
|--------|------------|
| Стулья офисные | На колёсиках (просто модель). |
| Настенные часы | Круглые, показывают время (стрелки статичны). |
| Лампы | Потолочные (люминесцентные) и настольные. |
| Картины | На стенах — абстракция или пейзажи в мультяшном стиле. |
| Ковры | Прямоугольные, цветные. |
| Растения в горшках | Просто для декора (пальмы, фикусы). |
| Огнетушитель | На стене в коридоре (красный). |
| Доска для объявлений | С бумажками, магнитами. |
| Вентилятор | Напольный или потолочный. |

### 2.4. Размеры и масштаб

- Офис: примерно **~200x200** (не слишком большой, чтобы охранник успевал бегать).
- Высота потолков: **~12-15** (чтобы не давило, но и не огромный ангар).
- Столы: высота **~2.5-3**, чтобы гоблин под ними помещался.

## 3. Общие требования по стилю

| Параметр | Требование |
|----------|------------|
| **Цветовая палитра** | Яркая, насыщенная. Офис — не серый, а с цветными стенами (например, бирюза, оранжевый, белый). Лобби — фиолетовый, синий, зелёный, розовый (магия). |
| **Освещение** | Желательно использовать **Voxel Lighting**. В офисах — тёплый свет от ламп. В лобби — сияние. |
| **Материалы** | Использовать гладкие поверхности. Без реалистичных шероховатостей. |
| **Пропорции** | Всё слегка преувеличено: большие мониторы, толстые столы, огромные кнопки на принтере. |

---

## 4. Требования к файлам

- Все модели должны быть **сгруппированы** в папки (Folder) в Workspace:
  - `Lobby`
  - `Office/Rooms` (по комнатам)
  - `Office/HidingSpots` (укрытия)
  - `Office/Tasks` (объекты для задач)
  - `Office/Decor`
- Коллизии (CanCollide) настроены правильно: мебель непроходимая, но под стол можно залезть (убрать коллизию снизу).
- Anchor все статичные объекты (чтобы не разлетались).
